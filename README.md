## Requirements:
* Electron
* Express
* StormDB
* EJS
* electron Packager
* NPX
* Code Editor


## Developement:

### Install Packages
<pre>
npm install
</pre>

### Starting app
<pre>
npm start
</pre>

### Building to ".exe"
<pre>
npm run package-win
</pre>
